# BidBoard

A simple full-stack auction-style application built with Java Spring Boot (backend) and Angular (frontend).

## Features (v1)
- Create users
- Create listings
- View listings
- Place bids
- View bid history
- Close auctions
- Enforce bid validation rules

## Tech Stack
Backend:
- Java 21
- Spring Boot
- Spring Data JPA
- H2 (dev) / PostgreSQL (later)

Frontend:
- Angular
- TypeScript

## Goal
This project is built to strengthen full-stack development skills and prepare for Software Engineering roles using Spring Boot and Angular.
